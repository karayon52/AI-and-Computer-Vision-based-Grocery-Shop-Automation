# AI-and-Computer-Vision-based-Grocery-Shop-Automation

1. download zip file and extract them.
2. put smartcart_stage3_autonomous_v2.py and best_april_3_small_model_on_custom.pt in the same folder.
3. get your Gemini API key from Google AI Studio
4. add your API key to the python script
5. connect microphone and 2 cameras: one for scaning and one for sorting
6. run esp32_code.ino in ArduinoIDE
7. after that run python script
8. then put your customer orders via either real time audio or scanner
9. then select autonomous sorting
